# yolox_Caffe_onnx 版本

yolox-nano_caffe_onnx版，后处理用python语言和C++形式进行改写，便于移植不同平台；为了在不同平台之间进行部署，对SiLU和Fuse进行了替换。

# 所需环境
1、用caffe版本demo需要caffe环境

2、用onnx版本demo需要python环境
